# Manual

## 输入目录

在 UI 的“选择目录”中导入 `work/inputs` 下按项目拆分的目录结构。

## 规则导入

在“规则”页上传规则 Excel，系统会写入：

- `work/inputs/rules.xlsx`
- `work/rules/rules.json`

## 扫描结果

扫描后只展示“不符合”项；每条不符合项支持展开查看证据区（文件 + 摘录）。

## 下载报告（Excel）

在“扫描结果”区域点击「下载报告(Excel)」，导出当前筛选后的不符合项列表。
